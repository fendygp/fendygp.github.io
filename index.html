Who I Am

Halo! Saya Fendy Gusta Pradana, seorang profesional IT dengan pengalaman kuat di industri perbankan. Saat ini saya menjabat sebagai Senior Manager Open API Platform Function Development di PT Bank Rakyat Indonesia (Persero) Tbk, di mana saya bertanggung jawab atas desain arsitektur sistem dan pengembangan produk API berbasis open banking.

Dengan lebih dari 8 tahun pengalaman di BRI, saya memiliki keahlian dalam pengembangan Web Application, API, dan microservices menggunakan Golang. Fokus utama saya adalah pada API Product & Gateway, Service-Oriented Architecture (SOA), dan Software Architectural Design yang mendukung transformasi digital perbankan.

Saya adalah lulusan Sarjana Teknik Informatika dari Universitas Brawijaya (2011-2016). Sebelum bergabung dengan BRI, saya memiliki pengalaman sebagai Full Stack Developer di berbagai perusahaan dan freelance projects. Saya percaya pada continuous learning, yang tercermin dari sertifikasi yang saya peroleh di bidang Software Testing dan Requirements Analysis.
